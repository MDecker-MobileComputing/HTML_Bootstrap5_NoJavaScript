Glossar mit Accordion von Bootstrap 5

Das folgende Glossar mit Einträgen zum Aufklappen wurde mit dem Accordion-Element von Bootstrap 5 erstellt. Durch einen Klick kann ein Eintrag aufgeklappt werden, wobei ein ggf. anderer geöffneter Eintrag zugeklappt wird.

Cascading Style Sheets. Deklarative Programmiersprache, mit der das Aussehen einer HTML-Seite definiert werden kann.

Hypertext Markup Language. Auszeichnungssprache, mit der die Struktur und der eigentliche Inhalt (v.a. Texte) einer Webseite definiert wird.

Hypertext Transfer Protocol. Ein zustandsloses Protokoll auf der Anwendungsschicht, das u.a. zur Übertragung der Dateien von einer Web-Seite vom Server zum Browser verwendet wird.

Programmiersprache. Mit JavaScript kann Code in eine Webseite eingebettet werden, der im Browser ausgeführt wird.

Uniform Resource Identifier Zeichenkette zur eindeutigen Definition einer Ressource; URLs sind ein Speziallfall von URIs.

Uniform Resource Locator Ein Speziallfall einer URI, der auch die Information beinhaltet, wo man die Ressource herbekommt. URLs sind das, was man in die Adresszeile eine Browsers normalerweise eingibt, um eine bestimmte Webseite aufzurufen.

Sammlung von Web-Seiten. Eine Website besteht aus einer oder mehreren Web-Seiten (HTML-Dokumente) und kann unter einer bestimmten URL (oftmals eine Domain) abgerufen werden.
